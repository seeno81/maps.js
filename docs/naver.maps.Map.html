Class: naver.maps.Map

naver.maps. Map

Map 클래스는 애플리케이션에서 지도 인스턴스를 정의합니다. 이 객체를 생성함으로써 개발자는 지정한 DOM 엘리먼트에 지도를 삽입할 수 있습니다.

new naver.maps.Map(mapDiv, mapOptions)

Parameters
Name Type Description
mapDiv string | HTMLElement

지도를 삽입할 HTML 엘리먼트 또는 HTML 엘리먼트의 id
mapOptions naver.maps.MapOptions

지도의 옵션 객체
Properties
Name Type Description
controls naver.maps.KVOArray

사전에 정의된 지도 내의 위치별로 지도 컨트롤의 인스턴스를 가지고 있는 객체입니다.

사용자 정의 컨트롤을 이 속성 내 특정 위치에 추가함으로써 사용자 정의 컨트롤을 지도에 추가할 수 있습니다.
data naver.maps.Data

데이터 레이어를 정의하는 Data 객체입니다.
layers LayerRegistry

지도 레이어의 컬렉션을 가지고 있는 객체입니다.
mapTypes naver.maps.MapTypeRegistry

지도 유형의 컬렉션을 가지고 있는 객체입니다.
mapSystemProjection naver.maps.MapSystemProjection

프로젝션 객체입니다.

지도 좌표와 세계 좌표, 화면 픽셀 좌표 간 좌표 변환할 수 있는 메서드를 제공합니다.

Type Definitions

MapPanes

MapPanes 객체는 오버레이를 지도 위에 렌더링할 수 있는 DOM 엘리먼트를 가지고 있으며, 지도에 있는 여러 레이어의 스택(stack) 순서를 나타냅니다.

Type:
  • object
Properties
Name Type Description
canvasTileLayer HTMLElement

이 페인은 타일 레이어 엘리먼트를 가집니다.
floatPane HTMLElement

이 페인은 모든 오버레이보다 위에 위치하는 정보 창(Info Window) 엘리먼트를 가집니다.
floatShadow HTMLElement

이 페인은 정보 창의 그림자 엘리먼트를 가집니다. 이 페인은 overlayImage 페인 위에 위치합니다. 따라서 마커는 정보 창 그림자에 의해 영향을 받을 수 있습니다.
overlayImage HTMLElement

이 페인은 마커의 이미지 엘리먼트를 가집니다.
overlayLayer HTMLElement

이 페인은 폴리라인, 폴리곤 등의 오버레이 엘리먼트를 가집니다. 이 페인은 DOM 이벤트를 받지 않습니다.
overlayMouseTarget HTMLElement

이 페인은 DOM 이벤트를 받을 수 있는 엘리먼트를 가집니다. 이 페인은 floatShadow 페인 위에 위치합니다. 따라서 정보 창 그림자에 있는 마커는 클릭할 수 있습니다.
overlayShadow HTMLElement

이 페인은 마커의 그림자 엘리먼트를 가집니다. 이 페인은 DOM 이벤트를 받지 않습니다.
