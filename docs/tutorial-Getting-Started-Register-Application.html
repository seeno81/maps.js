내 애플리케이션 등록하기

NAVER 지도 API v3를 사용하려면 클라이언트 아이디를 이용해서 인증해야 합니다. 클라이언트 아이디는 NAVER 개발자센터에서 애플리케이션을 등록해 발급받습니다.

NAVER 개발자센터에서 애플리케이션 등록 (API 이용신청)을 하려면 네이버 아이디가 필요합니다.

단계 1. 기본 정보 입력하기

NAVER 개발자센터의 애플리케이션 등록 (API 이용신청)에서 기본 정보를 입력합니다.

  • 애플리케이션 이름을 작성하고, 카테고리를 선택합니다.
  • 이용 목적은 비로그인 오픈API를 선택합니다.

단계 2. 웹 서비스 URL 등록하기

비로그인 오픈 API 서비스 환경 설정 화면이 열리면 오픈 API 이용 권한 신청의 지도를 체크합니다. 그리고 웹 서비스 URL란에 NAVER 지도 API v3를 사용하려는 사이트의 URL을 입력합니다.

웹 서비스 URL에 등록된 페이지에서만 NAVER 지도 API v3를 사용할 수 있습니다.

단계 3. 이용약관 확인하기

NAVER 지도 API v3 이용과 관련하여 권리, 의무 및 책임에 관한 제반 사항을 설명하고 있는 이용약관을 확인하고 등록하기 버튼을 클릭합니다.

단계 4. 발급받은 클라이언트 아이디 확인하기

모든 과정을 마치면 발급받은 클라이언트 아이디를 확인할 수 있습니다. Client ID에서 보여주는 클라이언트 아이디를 이용해 NAVER 지도 API v3를 로드할 수 있습니다.
